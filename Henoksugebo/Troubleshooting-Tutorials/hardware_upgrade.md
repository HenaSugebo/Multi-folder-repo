# README – hardware_upgrade.md

## 📄 File Overview
A practical tutorial for IT support officers on performing common hardware upgrades and replacements on desktops, laptops, and peripherals.

## 🎯 Purpose
- Standardise hardware maintenance procedures across the organisation.
- Reduce risk of component damage and data loss during upgrades.

## 🔧 Covered Procedures
1. **RAM Upgrade**
   - Compatibility check (DDR4 vs DDR5, speed, voltage)
   - ESD precautions and proper insertion technique
   - BIOS verification post‑install
2. **Storage Migration (HDD → SSD)**
   - Cloning with Macrium Reflect / Clonezilla
   - Adjusting boot order
   - Enabling AHCI mode
3. **Power Supply Unit (PSU) Replacement**
   - Wattage calculation and connector checks
   - Cable management tips
4. **Laptop Screen Replacement**
   - Safe bezel removal and hinge disassembly
   - Matching panel part numbers
   - Testing before final assembly
5. **Peripheral Add‑ons**
   - Installing a discrete GPU (power connectors, driver clean‑up)
   - Adding internal Wi‑Fi/Bluetooth cards on desktop
   - BIOS update procedures

## 🔒 Safety & Data Protection
- Always **back up critical data** before storage replacements.
- Power down and discharge capacitors (hold power button 10s after unplugging).
- Document serial numbers and asset tags updated in inventory system.

## 📸 Visual Aids
Document includes annotated photos of motherboard layouts, cable connectors, and BIOS screens for common enterprise models (Dell Optiplex, Lenovo ThinkPad).
